# 📱 手机书签方案 - 只看实时骑手监控

## 🔖 书签代码（一键复制）

**长按复制下面这行代码：**

```
javascript:(function(){document.querySelectorAll('.ant-layout-header,header,.partner-header').forEach(e=>e.style.display='none');document.querySelectorAll('.ant-layout-sider,aside,.ant-menu').forEach(e=>e.style.display='none');document.querySelectorAll('.ant-tabs-nav').forEach(nav=>{nav.querySelectorAll('.ant-tabs-tab').forEach(tab=>{const t=tab.textContent||'';if(!t.includes('实时')&&!t.includes('监控'))tab.style.display='none'})});document.querySelectorAll('.ant-layout-content,main').forEach(e=>{e.style.marginLeft='0';e.style.padding='0'});alert('已切换到实时监控模式！');})();
```

## 📲 iPhone (Safari) 设置步骤

### 第1步：先添加任意书签
1. 打开 Safari，访问任意网页（比如百度）
2. 点击底部 **分享按钮** ⬆️
3. 选择 **"添加书签"**
4. 名称写：`只看监控`
5. 点击 **存储**

### 第2步：编辑书签网址
1. 打开 Safari **书签**（底部书本图标📖）
2. 点击 **"编辑"**
3. 找到刚才添加的 `只看监控` 书签
4. 点击它，删除原来的网址
5. **粘贴上面复制的代码** 👆
6. 点击 **完成**

### 第3步：使用
1. 先打开美团管理后台并登录
2. 点击地址栏，输入 `只看监控`
3. 选择 `只看监控` 书签
4. 页面会自动刷新，只显示实时监控！

## 📲 安卓 (Chrome) 设置步骤

### 第1步：添加书签
1. 打开 Chrome，访问任意网页
2. 点击右上角 **⋮** 菜单
3. 选择 **"添加书签"** ⭐
4. 名称写：`只看监控`
5. 点击 **保存**

### 第2步：编辑书签
1. 点击 Chrome 地址栏右侧的 **⭐**
2. 点击 **"修改"** 或 **"编辑"**
3. 删除原来的网址
4. **粘贴上面复制的代码** 👆
5. 点击 **保存**

### 第3步：使用
1. 打开美团管理后台并登录
2. 点击地址栏
3. 输入 `只看监控`，选择该书签
4. 页面会切换到实时监控模式！

---

## ✅ 使用技巧

- **每次打开**美团后台，点一下书签即可
- 如果页面有变化，可能需要**刷新后再点**书签
- 书签可以添加到桌面，像 App 一样使用

## 🔄 添加到主屏幕（像 App 一样）

**iPhone:**
1. Safari 中点击分享按钮 ⬆️
2. 选择 **"添加到主屏幕"**
3. 点击 **添加**

**安卓:**
1. Chrome 中点击菜单 ⋮
2. 选择 **"添加到主屏幕"**
3. 点击 **添加**
